# 📊 Olist Brazilian E-Commerce Data Analysis & Power BI Dashboard

An end-to-end data analytics project on the **Olist Brazilian E-Commerce Dataset**, combining **SQL**, **PostgreSQL**, and **Power BI** to analyze sales performance, customer behavior, delivery efficiency, and payment trends.

---

##  Project Overview

This project explores the Brazilian e-commerce marketplace using real transactional data from Olist. The objective was to transform raw relational data into actionable business insights through SQL analysis and an interactive Power BI dashboard.

The workflow includes:

- Data Cleaning & Validation
- SQL Data Exploration
- Business KPI Analysis
- Dashboard Development
- Insight Generation

---

##  Tech Stack

- **SQL (PostgreSQL)**
- **Power BI**
- **CSV Dataset**
- **Data Modeling**
- **Business Intelligence**

---

## 📂 Dataset

The project uses the publicly available **Olist Brazilian E-Commerce Dataset**, containing information about:

- Customers

- Products

- Order Delivery

#  Dashboard Features

### Executive KPIs

- Total Orders
- Total GMV (Gross Merchandise Value)
- Average Customer Review Score

---

### Customer Analysis

- Customer distribution by customer type
- Interactive filtering by State and Year

---

### Sales Trend Analysis

- Monthly GMV Trend (2016–2018)
- Peak sales identification
- Seasonal sales fluctuations

---

### Delivery Performance

Average review scores based on delivery timing:

-  On Time
-  Slightly Late
-  Late (4–7 Days)
-  Very Late (7+ Days)
- Unknown

This helps understand how delivery impacts customer satisfaction.

---

### Payment Analysis

Payment value distribution across:

- Credit Card
- Boleto
- Voucher
- Debit Card
- Other

---

#  Key Insights

- Credit Cards account for the vast majority of revenue.
- Customers receiving orders on time leave significantly higher ratings.
- Monthly GMV shows strong growth throughout 2017 with seasonal spikes.
- Late deliveries correlate with lower customer satisfaction.
- Interactive filters allow regional and yearly performance comparison.
